## 2025 – Najważniejsze zmiany

### Claude 4 – Opus 4 i Sonnet 4.5 (2025)

Anthropic wypuścił rodzinę Claude 4. Sonnet 4.5 to model którego używam na co dzień – znacząca poprawa w kodowaniu i rozumowaniu w porównaniu do 3.5.

Opus 4 to najmocniejszy model Anthropic – do zadań gdzie liczy się jakość ponad wszystko.

### Claude Code – stabilna wersja (2025)

Claude Code przeszedł z beta do stabilnej wersji. To pełnoprawny agent kodowania w terminalu – nie chatbot z okienkiem.

### MCP staje się standardem (2025)

Model Context Protocol który Anthropic wypuścił jako standard integracji jest coraz szerzej adoptowany. Większość popularnych narzędzi (Notion, Slack, GitHub) ma teraz oficjalne serwery MCP.

### Web Search i upload plików – ulepszenia (2025)

Claude.ai dostał lepszy web search z cytowaniem źródeł i możliwość analizy większych plików PDF.
