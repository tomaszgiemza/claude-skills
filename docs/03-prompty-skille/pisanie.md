# Prompty – Pisanie i treści

## Artykuł blogowy SEO

```
Jesteś doświadczonym copywriterem SEO.

Napisz artykuł blogowy na temat: [TEMAT]
Słowo kluczowe: [FRAZA]
Długość: 1500 słów
Odbiorca: [OPIS ODBIORCY]
Ton: [przystępny/profesjonalny/ekspercki]

Struktura:
- Tytuł H1 z słowem kluczowym
- Wstęp (150 słów) – słowo kluczowe w 1. akapicie
- 4-5 sekcji H2 z podsekcjami H3
- FAQ (3 pytania)
- Podsumowanie z CTA

Zacznij od zaskakującego faktu lub pytania retorycznego.
```

## Opis produktu WooCommerce

```
Napisz opis produktu dla sklepu internetowego.

Produkt: [NAZWA PRODUKTU]
Kategoria: [KATEGORIA]
Główne cechy: [LISTA 3-5 CECH]
Cena: [CENA]
Odbiorca: [KTO KUPUJE]

Napisz:
1. Krótki opis (150 słów) – korzyści, emocje, CTA
2. Długi opis (400 słów) – szczegóły techniczne, FAQ
3. 5 bullet points z korzyściami
4. Meta opis (155 znaków)
```

## Przepisanie/ulepszenie tekstu

```
Ulepsz poniższy tekst zachowując jego znaczenie.

Cel: [co chcesz poprawić – np. "bardziej przekonujący", 
"prostszy język", "lepsze SEO", "krótszy o 30%"]

Oryginalny tekst:
[WKLEJ TEKST]

Podaj:
1. Ulepszoną wersję
2. Listę najważniejszych zmian które wprowadziłeś
```

## Post na LinkedIn

```
Napisz post na LinkedIn na temat: [TEMAT]

Format:
- Pierwsze zdanie: mocny hook (pytanie lub kontrowersja)
- 3-5 krótkich akapitów (1-2 zdania każdy)
- Lista 3 wniosków z emoji
- Zakończenie z pytaniem do dyskusji
- 3-5 hashtagów

Ton: profesjonalny ale osobisty
Długość: 200-300 słów
```

## Skrypt wideo/podcastu

```
Napisz skrypt do [wideo YouTube / odcinka podcastu] 
na temat: [TEMAT]

Długość: [X] minut (przyjmij ~130 słów/minutę)
Format: [wywiad / monolog / tutorial]

Struktura:
- Hook (pierwsze 15 sekund – dlaczego warto oglądać)
- Zapowiedź co będzie w odcinku
- Treść główna z płynnymi przejściami
- Podsumowanie i CTA (subskrypcja, komentarz)
```
