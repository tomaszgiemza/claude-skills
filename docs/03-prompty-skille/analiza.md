# Prompty – Analiza i research

## Analiza dokumentu / raportu

```
[Załącz plik PDF lub wklej treść]

Przeanalizuj ten dokument i:
1. Streszczenie (max 200 słów) – główne wnioski
2. Kluczowe dane liczbowe i fakty
3. Mocne strony / szanse
4. Słabe strony / zagrożenia (SWOT jeśli dotyczy)
5. 3 najważniejsze wnioski dla mnie jako [TWOJA ROLA]
6. Pytania które warto zadać po lekturze
```

## Research tematu

```
Zrób research na temat: [TEMAT]

Kontekst: [po co Ci ta wiedza]
Poziom: [podstawowy / średni / zaawansowany]

Potrzebuję:
1. Definicja i kluczowe pojęcia
2. Aktualny stan (trendy 2025)
3. Główne podejścia/metody/narzędzia
4. Zalety i wady każdego podejścia
5. Rekomendacja dla mojego przypadku: [KONTEKST]
6. Źródła / gdzie szukać więcej

Format: strukturowany raport z nagłówkami
```

## Analiza danych (CSV/tabela)

```
[Załącz plik CSV lub wklej tabelę danych]

Przeanalizuj te dane i:
1. Opisz co pokazują dane (ogólny obraz)
2. Zidentyfikuj trendy i wzorce
3. Wskaż anomalie lub ciekawe obserwacje
4. Oblicz: [średnia / mediana / suma / inne]
5. Porównaj: [które kolumny/okresy porównać]
6. Podaj 3-5 wniosków biznesowych
7. Zaproponuj kolejne kroki / rekomendacje

Wyniki podaj w formie: tekst + tabela podsumowująca
```

## Analiza SWOT / decyzja

```
Pomóż mi podjąć decyzję dotyczącą: [DECYZJA]

Kontekst:
- Sytuacja: [OPIS]
- Opcja A: [OPIS]
- Opcja B: [OPIS]
- Ograniczenia: [budżet, czas, zasoby]
- Cel: [co chcesz osiągnąć]

Przeprowadź analizę:
1. SWOT dla każdej opcji
2. Kryteria decyzyjne (zdefiniuj 5 najważniejszych)
3. Ocena opcji w skali 1-10 dla każdego kryterium
4. Rekomendacja z uzasadnieniem
5. Ryzyka i jak je mitygować
```

## Podsumowanie artykułu/wideo

```
Podsumuj poniższy [artykuł/transkrypt wideo/podcast]:

[WKLEJ TREŚĆ lub URL]

Format podsumowania:
- TL;DR (3 zdania max)
- Główne punkty (5-7 bullet points)
- Kluczowe cytaty (2-3)
- Moje zastosowanie: co mogę z tego wykorzystać w [KONTEKST]
- Ocena wartości: czy warto polecić? dlaczego?
```
