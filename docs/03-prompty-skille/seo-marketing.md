# Prompty – SEO i marketing

## Badanie słów kluczowych

```
Jesteś ekspertem SEO specjalizującym się w polskim rynku.

Dla tematu: [TEMAT/BRANŻA]
Strona: [URL lub opis strony]

Zaproponuj:
1. 10 słów kluczowych głównych (head keywords) z szacowanym wolumenem
2. 20 long tail keywords (3-5 słów) z niską konkurencją
3. 10 pytań które zadają użytkownicy (do FAQ/featured snippets)
4. 5 tematów na artykuły które mogą wypozycjonować się szybko

Pogrupuj słowa kluczowe tematycznie.
```

## Meta tytuł i opis

```
Napisz zoptymalizowane meta tagi dla strony:

Temat strony: [OPIS]
Główne słowo kluczowe: [FRAZA]
Typ strony: [artykuł / strona produktu / strona kategorii / strona główna]

Napisz 3 warianty:
- Meta title (50-60 znaków) – słowo kluczowe na początku
- Meta description (150-160 znaków) – korzyść + CTA

Zaznacz najlepszy wariant i uzasadnij wybór.
```

## Analiza artykułu pod SEO

```
Przeanalizuj poniższy artykuł pod kątem SEO.
Główne słowo kluczowe: [FRAZA]

[WKLEJ ARTYKUŁ]

Oceń:
1. Użycie słowa kluczowego (tytuł, nagłówki, wstęp, gęstość)
2. Strukturę nagłówków H1-H4
3. Alt texty (zasugeruj treść)
4. Linki wewnętrzne (których brakuje)
5. Długość treści
6. Czytelność (zdania, akapity)

Podaj listę konkretnych poprawek z priorytetem: WYSOKI/ŚREDNI/NISKI
```

## Strategia content marketingu

```
Stwórz 3-miesięczną strategię content marketingu dla:

Firma/blog: [OPIS]
Branża: [BRANŻA]
Cel: [np. "zwiększyć ruch o 50%", "pozyskać leady B2B"]
Zasoby: [np. "1 artykuł/tydzień, budżet 0"]

Strategia powinna zawierać:
- Podział na miesiące z celami
- Tematy artykułów (minimum 12)
- Typy treści (artykuły, infografiki, case studies)
- Dystrybucja (social media, newsletter, link building)
- KPI do śledzenia
```

## Analiza konkurencji

```
[Załącz zrzuty ekranu lub wklej URL konkurenta]

Przeanalizuj stronę konkurenta:
1. Jakie słowa kluczowe targetuje (na podstawie treści)?
2. Jaka jest struktura nawigacji?
3. Jakie typy treści publikuje?
4. Jakie są ich USP (unique selling points)?
5. Jakie luki możesz wykorzystać?

Podaj 5 konkretnych przewag które możesz zbudować.
```
