# Prompty – Email i komunikacja

## Email biznesowy / oferta

```
Napisz profesjonalny email biznesowy.

Cel: [oferta / follow-up / podziękowanie / prośba]
Od: [Twoja rola i firma]
Do: [kto jest odbiorcą]
Kontekst: [co było wcześniej / background]
Główny przekaz: [co chcesz osiągnąć]
Ton: [formalny / półformalny / przyjazny]

Email powinien:
- Mieć chwytliwy temat (subject line)
- Być zwięzły (max 200 słów)
- Mieć jasne CTA na końcu
```

## Odpowiedź na reklamację

```
Napisz profesjonalną odpowiedź na reklamację klienta.

Reklamacja klienta:
[WKLEJ TREŚĆ REKLAMACJI]

Sytuacja:
- Co się stało: [OPIS]
- Nasza wina / niezależne: [OPIS]
- Co możemy zaoferować: [ROZWIĄZANIE]

Odpowiedź powinna:
- Przeprosić (jeśli nasza wina) lub wyjaśnić
- Pokazać empatię
- Zaproponować konkretne rozwiązanie
- Zakończyć pozytywnie
- Zachować profesjonalny ton
```

## Negocjacje / prośba o rabat

```
Napisz email z prośbą o [rabat / lepsze warunki / przedłużenie terminu].

Dostawca/partner: [FIRMA]
Obecne warunki: [OPIS]
Czego chcesz: [KONKRETNA PROŚBA]
Argumenty: [DLACZEGO POWINIENEŚ DOSTAĆ]
Relacja: [nowy klient / stały klient od X lat]

Ton: asertywny ale uprzejmy, nie agresywny
```

## Newsletter / email marketingowy

```
Napisz email do newslettera.

Lista: [OPIS SUBSKRYBENTÓW]
Temat odcinka: [TEMAT]
Cel emaila: [informacja / sprzedaż / angażowanie]
Główna wartość dla czytelnika: [CO DOSTAJĄ]

Struktura:
- Subject line (3 warianty do A/B testu)
- Preview text (90 znaków)
- Treść (300-400 słów)
- CTA przycisk (tekst)

Ton: [TWÓJ TON KOMUNIKACJI]
```

## Podsumowanie spotkania / MoM

```
Na podstawie poniższych notatek ze spotkania 
napisz formalne podsumowanie (Minutes of Meeting).

Notatki:
[WKLEJ NOTATKI]

Format:
- Data, uczestnicy
- Omówione tematy (bullet points)
- Podjęte decyzje
- Action items (kto / co / do kiedy)
- Następne kroki
```
