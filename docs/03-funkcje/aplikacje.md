# Claude w aplikacjach

Anthropic wypuścił kilka beta produktów które integrują Claude z popularnymi aplikacjami.

{: .warning }
Te funkcje są w **beta** – mogą się zmieniać. Sprawdź aktualny status na [claude.ai](https://claude.ai).

## Claude in Chrome

Rozszerzenie do przeglądarki Chrome które sprawia że Claude "widzi" to co Ty na ekranie.

**Co potrafi:**
- Podsumować aktualnie otwartą stronę
- Wytłumaczyć zaznaczony tekst
- Pomóc wypełnić formularz
- Odpowiedzieć na pytania dotyczące zawartości strony

Instalacja: Chrome Web Store → szukaj "Claude for Chrome"

## Claude in Excel

Integracja z Microsoft Excel – Claude jako asystent do arkuszy.

**Co potrafi:**
- Pisać formuły na podstawie opisu słownego
- Analizować dane i tworzyć podsumowania
- Wyjaśniać istniejące formuły
- Sugerować wizualizacje

## Claude in PowerPoint

Pomoc przy tworzeniu prezentacji.

**Co potrafi:**
- Tworzyć strukturę prezentacji z briefu
- Pisać treść slajdów
- Sugerować układ i design
- Podsumować istniejącą prezentację

## Cowork – automatyzacja plików

Desktop tool do automatyzacji zadań związanych z plikami i folderami. Beta, dla nie-deweloperów.

{: .highlight }
**Moje zdanie:** Te integracje są obiecujące ale wciąż dojrzewają. Claude in Chrome jest najbardziej użyteczny już teraz. Reszta – obserwuję i czekam na stabilniejsze wersje.
