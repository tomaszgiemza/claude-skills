# Prompty – Biznes

## Oferta handlowa

```
Napisz profesjonalną ofertę handlową.

Moja firma: [NAZWA, branża]
Produkt/Usługa: [OPIS]
Klient: [KIM JEST – firma, branża, wielkość]
Potrzeba klienta: [CO ZGŁOSIŁ / JAKI MA PROBLEM]
Cena: [WARTOŚĆ lub "do ustalenia"]
Termin realizacji: [TERMIN]
Wyróżniki: [DLACZEGO MY, nie konkurencja]

Struktura:
- Streszczenie wykonawcze (3 zdania)
- Rozumienie potrzeby klienta
- Proponowane rozwiązanie
- Zakres prac / deliverables
- Harmonogram
- Cena (z opcjami jeśli dotyczy)
- Dlaczego my – 3 konkretne argumenty
- Następny krok / CTA

Format: profesjonalny, bez nadmiaru buzzwordów.
```

## Analiza SWOT

```
Przeprowadź analizę SWOT dla:

Firma/projekt: [OPIS]
Branża: [BRANŻA]
Kontekst: [np. wejście na nowy rynek / nowy produkt / fuzja / restrukturyzacja]

Dla każdej ćwiartki podaj 4-6 konkretnych punktów (nie ogólników).

Na końcu:
- 3 strategiczne rekomendacje wynikające z analizy
- Główne ryzyko i jak je mitygować
- Szansa której nie możesz przegapić
```

## Email negocjacyjny

```
Napisz email negocjacyjny.

Sytuacja: [KONTEKST – z kim negocjujesz, co jest stawką]
Moja pozycja: [CO CHCĘ OSIĄGNĄĆ]
Ich pozycja: [CO WIEM O ICH INTERESACH]
Moje argumenty: [DLACZEGO MOJA PROŚBA JEST UZASADNIONA]
Relacja: [nowy kontakt / długoletni partner / dostawca]
BATNA (co zrobię jeśli nie dogadam): [OPCJA ALTERNATYWNA]

Styl: asertywny, oparty na faktach, nie agresywny.
Zakończ konkretną propozycją i pytaniem.
```

## Prezentacja biznesowa – struktura i treść

```
Stwórz strukturę i treść prezentacji biznesowej.

Cel prezentacji: [np. pitch inwestorski / raport zarządu / oferta dla klienta]
Odbiorca: [KIM SĄ – co wiedzą, czego oczekują, co ich motywuje]
Czas: [X minut]
Główny przekaz (1 zdanie): [CO CHCESZ ŻEBY ZAPAMIĘTALI]

Dla każdego slajdu:
- Tytuł slajdu
- Główna teza (1 zdanie)
- 3 kluczowe punkty / dane
- Sugestia wizualizacji

Maksymalnie [X] slajdów.
```

## Analiza danych – raport zarządu

```
[Wklej dane lub opis sytuacji]

Napisz raport dla zarządu (max 1 strona A4).

Struktura:
1. Podsumowanie wykonawcze (3 zdania, najważniejszy wniosek na początku)
2. Kluczowe liczby (tabela lub 4-6 metryk z kontekstem vs poprzedni okres)
3. Co działało dobrze
4. Co wymaga uwagi
5. Rekomendacje (3 konkretne działania z właścicielem i terminem)

Pisz dla osób które mają 5 minut i potrzebują decyzji, nie szczegółów.
```
