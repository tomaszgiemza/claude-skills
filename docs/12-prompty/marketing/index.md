# Prompty – Marketing

## Strategia kampanii Google Ads

```
Stwórz strategię kampanii Google Ads.

Produkt/Usługa: [OPIS]
Budżet miesięczny: [KWOTA PLN]
Cel: [np. sprzedaż / leady / ruch / świadomość]
Target: [OPIS ODBIORCY – demografia, zainteresowania, intencja]
Lokalizacja: [GEOGRAFICZNIE]
Konkurencja: [czy jest duża?]

Zaproponuj:
1. Typ kampanii (Search / Shopping / Display / Performance Max)
2. Struktura konta (kampanie → grupy reklam)
3. 10 głównych słów kluczowych z typem dopasowania
5. Wykluczające słowa kluczowe (minimum 10)
6. Strategię ustalania stawek
7. Metryki sukcesu (KPI)
```

## Reklamy Meta (Facebook/Instagram)

```
Napisz kreacje reklamowe do Meta Ads.

Produkt/Usługa: [OPIS]
Cel kampanii: [świadomość / ruch / konwersja / leady]
Target audience: [OPIS – wiek, płeć, zainteresowania, zachowania]
USP (wyróżnik): [CO CIĘ WYRÓŻNIA]
Budżet: [DZIENNY / MIESIĘCZNY]

Dla każdego formatu:

**Single Image Ad:**
- Headline (max 40 znaków)
- Primary text (max 125 znaków przed "Więcej")
- CTA button: [wybierz z listy Meta]

**Story/Reel Ad (wideo):**
- Hook (pierwsze 3 sekundy – co pokazać/powiedzieć)
- Główny przekaz (5-10 sekund)
- CTA (ostatnie 2 sekundy)

Napisz 3 różne warianty każdego formatu do testów A/B.
```

## Email marketing – sekwencja powitalna

```
Napisz sekwencję powitalną dla nowych subskrybentów.

Produkt/Usługa/Blog: [OPIS]
Cel sekwencji: [budowanie relacji / sprzedaż / edukacja]
Liczba emaili: 5
Harmonogram: email 1 (od razu), email 2 (dzień 3), email 3 (dzień 7), email 4 (dzień 14), email 5 (dzień 21)

Dla każdego emaila:
- Subject line (3 warianty)
- Preview text (max 90 znaków)
- Cel tego emaila (1 zdanie)
- Główna treść (150-250 słów)
- CTA

Tone of voice: [OPISZ SWÓJ STYL]
```

## Lejek sprzedażowy – mapowanie

```
Zaprojektuj lejek sprzedażowy.

Produkt/Usługa: [OPIS]
Cena: [WARTOŚĆ]
Cykl zakupowy: [jak długo klient decyduje się na zakup]
Główne obiekcje klientów: [LISTA]

Zaprojektuj lejek:

TOFU (świadomość):
- Treści przyciągające ruch (typy, tematy)
- Kanały dotarcia

MOFU (rozważanie):
- Lead magnet (co oferujesz za email)
- Treści budujące zaufanie
- Email nurturing (5 emaili – tematy)

BOFU (decyzja):
- Strona sprzedażowa – kluczowe elementy
- Oferty specjalne / FOMO
- Obiekcje i jak je zbijać

Dla każdego etapu: konkretne działania, nie ogólniki.
```

## Analiza kampanii marketingowej

```
[Wklej dane kampanii: CTR, konwersje, koszty, etc.]

Przeanalizuj wyniki kampanii marketingowej.

Cel kampanii był: [CEL]
Oczekiwania: [CO PLANOWAŁEŚ OSIĄGNĄĆ]

Oceń:
1. Czy cel został osiągnięty? O ile procent?
2. Które elementy działały najlepiej (kreacje, grupy docelowe, słowa kluczowe)
3. Co przeciągało wyniki w dół
4. Anomalie i co je mogło spowodować
5. 3 konkretne zmiany na następną kampanię

Format: executive summary (3 zdania) + szczegółowa analiza.
```
