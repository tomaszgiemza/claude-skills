# Prompty – SEO

## Badanie słów kluczowych

```
Przeprowadź badanie słów kluczowych dla:

Strona/branża: [OPIS]
Główny temat: [TEMAT]
Lokalizacja (jeśli lokalne SEO): [MIASTO/REGION]
Konkurenci (jeśli znasz): [LISTA]

Podaj:
1. 10 słów kluczowych głównych (head keywords) z szacowaną trudnością: niska/średnia/wysoka
2. 20 long tail keywords (3-5 słów) – skup się na niskiej konkurencji
3. 10 pytań które zadają użytkownicy (do FAQ i featured snippets)
4. 5 tematów na artykuły które możesz wypozycjonować w 3-6 miesięcy

Pogrupuj tematycznie. Zaznacz TOP 5 do zaczęcia.
```

## Meta tytuł i opis

```
Napisz zoptymalizowane meta tagi dla:

Typ strony: [artykuł / strona produktu / kategoria / strona główna]
Temat: [OPIS TREŚCI]
Główne słowo kluczowe: [FRAZA]
Tonacja: [informacyjna / sprzedażowa / lokalna]

Napisz 3 warianty (każdy inny kąt):
- Meta title: 50-60 znaków, słowo kluczowe na początku
- Meta description: 145-160 znaków, słowo kluczowe, korzyść, CTA

Podaj: licznik znaków dla każdego i rekomendację który wybrać.
```

## Audyt SEO strony

```
[Wklej URL lub opis strony / fragment treści]

Przeprowadź audyt SEO i oceń:

On-page:
- Tytuł H1 (czy zawiera słowo kluczowe, długość)
- Struktura nagłówków H2-H4
- Użycie słowa kluczowego (gęstość, miejsca)
- Meta opis (czy jest, długość, słowo kluczowe)
- Alt texty obrazów
- Linkowanie wewnętrzne
- Długość treści vs konkurencja

Technicznie:
- URL (struktura, długość, słowo kluczowe)
- Czy treść odpowiada intencji wyszukiwania?

Podaj: ocena 1-10, lista krytycznych błędów, lista ulepszeń z priorytetem.
```

## Strategia content marketingu

```
Stwórz 3-miesięczną strategię treści dla:

Branża/niche: [BRANŻA]
Cel: [np. zwiększyć ruch o 50% / pozyskać leady / budować autorytet]
Zasoby: [np. 2 artykuły/tydzień, brak budżetu na linki]
Aktualna sytuacja: [np. nowa strona / 50 artykułów / 1000 wizyt/mies]
Konkurencja: [poziom: niska/średnia/wysoka]

Plan miesięczny:
- Miesiąc 1: Quick wins (łatwe do wypozycjonowania)
- Miesiąc 2: Budowanie autorytetu tematu
- Miesiąc 3: Treści linkbuildingowe

Dla każdego miesiąca: 4-6 tematów artykułów z słowem kluczowym i typem treści.
```

## Linkowanie wewnętrzne

```
[Wklej listę URL lub tytułów artykułów na swojej stronie]

Zaproponuj strategię linkowania wewnętrznego:

1. Zidentyfikuj pillar pages (główne kategorie tematyczne)
2. Pogrupuj artykuły w klastry tematyczne
3. Dla każdego artykułu – do których 2-3 innych artykułów powinien linkować
4. Jakie anchor texty używać (naturalne, z słowem kluczowym)

Cel: zbudowanie silosu tematycznego który wzmocni autorytet kategorii.
```
