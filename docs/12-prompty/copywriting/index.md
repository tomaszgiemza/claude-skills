# Prompty – Copywriting

## Opis produktu – sklep online

```
Napisz opis produktu do sklepu internetowego.

Produkt: [NAZWA PRODUKTU]
Kategoria: [np. elektronika / odzież / kosmetyki]
Główne cechy: [LISTA 3-5 CECH]
Cena: [CENA]
Odbiorca: [KTO KUPUJE – np. "profesjonalni fotografowie"]
USP (wyróżnik): [CO ODRÓŻNIA OD KONKURENCJI]

Napisz:
1. Krótki opis (max 150 słów) – emocje, korzyści, CTA
2. Długi opis (350-500 słów) – szczegóły, zastosowania, FAQ
3. 5 bullet points z kluczowymi korzyściami (nie cechami!)
4. Meta opis SEO (max 155 znaków)
```

## Landing page – hero section

```
Napisz teksty do sekcji hero na landing page.

Produkt/Usługa: [OPIS]
Główna korzyść dla klienta: [KORZYŚĆ]
Problem który rozwiązujesz: [PROBLEM]
Odbiorca: [KTO TO CZYTA]
Tone of voice: [np. ekspercki / przyjazny / motywujący]

Napisz 3 warianty:
- Nagłówek H1 (max 8 słów)
- Podtytuł (max 20 słów)
- Przycisk CTA (max 5 słów)

Każdy wariant podkreśla inny aspekt (korzyść / problem / ciekawość).
```

## Newsletter – odcinek

```
Napisz odcinek newslettera na temat: [TEMAT]

Lista: [OPIS SUBSKRYBENTÓW – co ich łączy]
Cel odcinka: [informacja / sprzedaż / angażowanie / edukacja]
Główna wartość dla czytelnika: [CO DOSTAJĄ]
Długość: 300-400 słów

Struktura:
- Subject line (3 warianty do A/B testu)
- Preview text (max 90 znaków)
- Hook (pierwsze zdanie – zatrzymuje od scrollowania)
- Treść główna
- CTA (jedno, konkretne)

Ton: [TWÓJ TON – np. bezpośredni, przyjacielski, ekspercki]
```

## Post na LinkedIn – thought leadership

```
Napisz post LinkedIn na temat: [TEMAT]

Kąt (angle): [np. "kontrowersja", "lekcja z błędu", "nieoczekiwane odkrycie"]
Moje doświadczenie w tym: [1-2 zdania kontekstu]
Główny wniosek: [CO CHCESZ ŻEBY ZAPAMIĘTALI]

Format:
- Linia 1: mocny hook (pytanie / liczba / kontrowersja)
- Linie 2-8: rozwinięcie w krótkich akapitach
- Linie 9-12: lista wniosków z emoji ✅
- Ostatnia linia: pytanie do dyskusji
- 3-5 hashtagów

Długość: 200-300 słów
```

## Reklama Google Ads

```
Napisz reklamy tekstowe Google Ads.

Produkt/Usługa: [OPIS]
Słowo kluczowe: [GŁÓWNA FRAZA]
Landing page: [URL lub opis]
USP: [CO CIĘ WYRÓŻNIA]
CTA: [np. "Kup teraz", "Sprawdź cennik", "Umów się"]

Napisz 3 zestawy:
- Nagłówek 1 (max 30 znaków) – słowo kluczowe
- Nagłówek 2 (max 30 znaków) – USP
- Nagłówek 3 (max 30 znaków) – CTA
- Opis (max 90 znaków)

Zaznacz który zestaw polecasz i dlaczego.
```
