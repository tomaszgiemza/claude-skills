# Interfejs Claude.ai

## Elementy interfejsu

### Panel boczny (lewy)
- **Nowa rozmowa** – przycisk `+` lub skrót `Ctrl+Shift+O`
- **Historia rozmów** – lista poprzednich konwersacji
- **Projects** – grupy rozmów z pamięcią i instrukcjami
- **Starred** – przypięte ważne rozmowy

### Pole wiadomości (dół)
- Wpisywanie tekstu
- 📎 Załączanie plików (PDF, obrazy, kod, CSV)
- 🌐 Web search – przeszukiwanie internetu
- Wybór modelu (Sonnet / Opus / Haiku)

### Pasek górny
- Nazwa bieżącej rozmowy (kliknij żeby zmienić)
- Ustawienia konta

## Skróty klawiaturowe

| Skrót | Akcja |
|-------|-------|
| `Enter` | Wyślij wiadomość |
| `Shift+Enter` | Nowa linia bez wysyłania |
| `Ctrl+Shift+O` | Nowa rozmowa |
| `Ctrl+K` | Wyszukaj w historii |
| `Esc` | Zatrzymaj generowanie |

## Ustawienia (Settings)

Ikona profilu → Settings:
- **Appearance** – jasny/ciemny motyw
- **Memory** – zarządzanie pamięcią Claude
- **Profile** – imię i preferencje które Claude zapamiętuje
- **Privacy** – ustawienia danych

{: .note }
Wypełnij **Profile** – powiedz Claude kim jesteś i jak chcesz żeby do Ciebie mówił. To znacznie poprawia jakość odpowiedzi.
