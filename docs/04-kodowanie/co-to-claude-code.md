# Claude Code – co to jest?

Gdy po raz pierwszy uruchomiłem Claude Code w terminalu, przez chwilę siedziałem i patrzyłem w ekran. To nie jest chatbot z okienkiem. To coś innego.

## Czym jest Claude Code?

Claude Code to **agent AI w terminalu**. Działa bezpośrednio w Twoim środowisku pracy – widzi pliki, uruchamia komendy, czyta logi, pisze kod i od razu go testuje.

### Czat vs Claude Code – kluczowa różnica

| Claude.ai (czat) | Claude Code (terminal) |
|------------------|----------------------|
| Wklejasz kod ręcznie | Widzi wszystkie pliki projektu |
| Dostajesz kod do skopiowania | Sam pisze i zapisuje pliki |
| Testujesz ręcznie | Sam uruchamia i testuje |
| Kontekst jednej rozmowy | Pamięta cały projekt |
| Przez przeglądarkę | W terminalu / VS Code |

### Co Claude Code potrafi zrobić

Powiedz mu: *"Dodaj do tego sklepu WooCommerce możliwość filtrowania produktów po kolorze i dodaj to do strony kategorii"*

Claude Code:
1. Przejrzy Twoje pliki projektu
2. Zrozumie strukturę motywu i używane hooki
3. Napisze kod PHP, CSS i JS
4. Doda odpowiednie pliki
5. Poinformuje co zmienił i gdzie

Bez kopiowania, wklejania, ręcznego tworzenia plików.

## Kiedy używać Claude Code a kiedy czatu?

**Użyj Claude Code gdy:**
- Pracujesz nad projektem z wieloma plikami
- Potrzebujesz wielokrokowych zmian w kodzie
- Chcesz żeby Claude sam uruchamiał i testował
- Chcesz mieć agenta który działa autonomicznie

**Użyj czatu gdy:**
- Masz szybkie pytanie o kod
- Chcesz wyjaśnienia jak coś działa
- Nie chcesz dawać dostępu do plików
- Pracujesz na urządzeniu mobilnym
